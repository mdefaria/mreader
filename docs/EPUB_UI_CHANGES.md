## Key Visual Changes Summary

1. **Uploader Message**: Updated to mention "text or EPUB file"
2. **File Acceptance**: Shows .epub in accepted formats
3. **File Size Limit**: Increased to 50MB (displayed in uploader)
4. **EPUB Badge**: Small red badge appears next to EPUB book titles
5. **Author Display**: More likely to show author name (extracted from EPUB metadata)
6. **No Change to Reader**: Reading experience identical for all formats
